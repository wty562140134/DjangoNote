# DjangoNote
## 1.安装Django:<br>
    pip install django==<version><br>
## 2.创建工程：<br>
    django-admin.py startporject <porject_name> <dir_path>
   或<br>
   
    django-admin startporject <porject_name> <dir_path>
## 3.创建应用：<br>
    python manage.py <app_name>
## 4.运行工程：<br>
    python manage.py runserver <port>
